## نبذة

موقعي الشخصي ومعرض أعمالي — مبني بـ HTML و CSS و JavaScript خالص، بدون أي إطار عمل، ومستضاف على GitHub Pages. يعرض مشاريعي عبر محاكي متصفح حيّ، خدماتي، وأداة أمنية تفاعلية، مع نموذج لاستقبال طلبات المشاريع مباشرة.

## المميزات

- 🖥️ **محاكي متصفح حيّ** يتنقّل بين المشاريع الحقيقية
- 🛡️ **فاحص روابط أمني** تفاعلي (تحليل محلي + تكامل VirusTotal)
- 📨 **نموذج استقبال مشاريع** يصل مباشرة عبر البريد
- 🎨 هوية بصرية فخمة بخلفية متحركة وتصميم متجاوب بالكامل
- ⚡ سريع، خفيف، ومهيّأ لمحركات البحث (SEO)

## الخدمات

| الخدمة | الوصف |
